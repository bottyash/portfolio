# AI-Native Portfolio

An AI-powered interactive portfolio built with Next.js and Mistral AI.

**Live:** [yashkparmar.vercel.app](https://yashkparmar.vercel.app)

## Setup

### Prerequisites
- Node.js v18+
- Mistral API key
- GitHub personal access token

### Install & Run

```sh
git clone https://github.com/bottyash/ai-native-portfolio.git
cd ai-native-portfolio
npm install --legacy-peer-deps
```

Create `.env.local`:

```env
MISTRAL_API_KEY="your_key_here"
GITHUB_TOKEN="your_token_here"
```

Start dev server:

```sh
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Tech Stack

- Next.js 15, React, Tailwind CSS, Framer Motion
- Mistral AI API
- Vercel (deployment)
